# Magic Clean - Landing Page

Site institucional da **Magic Clean Orlando** — serviço de limpeza 5 estrelas em Orlando, FL.

## Stack

- **React 18** + **TypeScript**
- **Vite** (build)
- **Tailwind CSS** + **shadcn/ui**
- **Framer Motion** (animações)
- **i18n** customizado (PT 🇧🇷 / EN 🇺🇸 / ES 🇪🇸)

## Rodando localmente

```bash
npm install
npm run dev
```

## Deploy para GitHub Pages

```bash
npm run build
# commit e push do conteúdo de dist/ para a branch gh-pages
git add dist -f
git commit -m "Deploy"
git subtree push --prefix dist origin gh-pages
```
